# Database Setup

## Clone the repository
```
git clone https://github.com/mx-kimber/cloudinary-tutorial-rails.git
```
## Install dependencies 
```
bundle install
```
## Create the database
```
rails db:create
```

## Set up the database

```
rails db:migrate
rails db:seed 
```
